# python-libsnake
A small snake game library for Python x Pygame.

# requirements:
pygame module.

# usage
place `snake.py` and `food.py` in the same directory as your `insertnameofyourmainpyfilehere.py`
then add these three lines to the top of your file:
```
import pygame
from snake import *
from food import *
```
